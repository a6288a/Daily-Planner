#每日规划笔记本（每日规划笔记本）

![许可证] ( https://img.shields.io/badge/license-MIT-blue.svg )
![版本] ( https://img.shields.io/badge/version-1.0.0-green.svg )

一个简单而强大的个人时间管理工具，帮助您更好地规划和记录每一天。使用纯前端技术构建，云端服务器即可运行。

## ✨ 特性

- 📝 实时时间轴记录
- 🎯“三只青蛙”任务管理系统
- 🔄支持撤销操作
- 💾 本地数据存储
- 📱 响应式设计，支持移动端
- 🎨 优雅的动画效果

## 💡使用指南

### 时间轴功能
- 点击时间轴空白处添加新记录
- 实时记录重要事项、会议、想法等
-支持编辑和删除操作

### 三只青蛙任务
- 在右侧设置每日三个最重要的任务
- 建议在早上计划，优先完成
- 可以修改随时任务内容

### 数据管理
- 所有数据安全存储在本地
- 支持撤销最近的操作
- 保护用户隐私，数据不会上传

## 🛠️ 技术栈

- **扩展框架**：扩展HTML5 + CSS3 + JavaScript
- **存储方案**：LocalStorage
- ** UI/UX **：响应式设计 + CSS 动画
